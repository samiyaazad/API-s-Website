## TECHNOLOGIES [THAT WERE USED]

        - HTML
        - CSS
        - JavaScript
        - VS Code


## Features 

This web app fetches the Astronomy Picture of the Day (APOD) from NASA’s API and displays:
                                                                                          - The image (as background)
                                                                                          - The date
                                                                                          - Explanation
                                                                                          - A clickable title linking to the HD version


## PROCESS

                      Step 1: HTML Structure
                      Step 2: CSS Styling
                      Step 3: Connect API
                      Step 4: Fetch Data
                      Step 5: Display Data
                      Step 6: Handle Special Cases


## What I learned 

              1.  What an API is
              2.  Fetching Data (JavaScript)
              3.  DOM Manipulation
              4.  Dynamic Styling
